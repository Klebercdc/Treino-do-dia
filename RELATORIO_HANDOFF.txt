1. FONTES LEGADAS MANTIDAS NO IMPORT
   Mantivemos DM Sans/DM Mono/Bebas Neue no import porque ha trechos de codigo
   JS que renderizam elementos com essas fontes via style inline. Remoção do
   import quebraria esses casos. Chakra Petch e IBM Plex Mono foram ADICIONADOS.

2. ex-title: font-style italic removido
   Era `font-style: italic` com Bebas Neue. Chakra Petch italic existe no import,
   mas mantivemos upright para maior legibilidade no novo estilo. Facil de reverter.

3. Dieta choice card --orange: icone virou roxo (#6B4EFF) em vez de azul
   A spec dizia azul para ambos os cards. Usei azul para "Dieta IA" e roxo
   (--secondary) para "Montar Manual" para diferenciacao visual. 1 linha para unificar.

4. .home-hero-icon--green: usa #7fb0ff em vez de #0055FF
   Leve diferenca entre card Treino e Dieta. Facil de unificar se preferir.

5. Texture/ruido (::before no body) NAO adicionado
   A spec marcava como "opcional". Nao adicionei para evitar risco de z-index
   conflitando com elementos fixos. Pode ser adicionado isoladamente.

6. Light mode NAO alterado
   Todos os novos estilos ficam no default (dark). O light mode mantem sua
   paleta laranja via `body.light-mode { --accent: #ea6c0a }` existente.
