# Identidade: KRONIA TRANSFORMS (Stark Mode / iOS Native)

## 1. Visão Geral
Aplicativo móvel de alta performance focado em gestão de carga e prevenção de lesões.
O assistente KRONOS atua como um "J.A.R.V.I.S. da musculação", processando o Nó FAD (Fadiga) e gerando o Nó REC (Recomendação).

## 2. Stack Técnica
- Front-end: React Native (Mobile Nativo)
- Back-end/DB: Supabase (PostgreSQL)
- Estilo: iOS Dark Mode (Referência: WhatsApp iOS / Stark Industries)
- Motor de IA: Algoritmo ACWR (Acute:Chronic Workload Ratio)
