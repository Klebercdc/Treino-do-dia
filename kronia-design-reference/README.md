# KRONIA Design Reference

Esta pasta é a fonte de referência visual para replicar o layout do design enviado no app real.

## Objetivo

Replicar o estilo do design de referência nas telas existentes do KRONIA, mantendo as funcionalidades reais funcionando.

## Fonte principal

O arquivo abaixo deve ser usado como referência principal:

`kronia-design-reference/source/KroniA-designer.zip`

Ele contém telas, componentes e assets visuais, incluindo imagens anatômicas/corporais 3D.

## Direção obrigatória

- Replicar o estilo visual do ZIP com alta fidelidade.
- Adaptar somente para as telas reais já existentes no app.
- Não criar novo aplicativo.
- Não adicionar funcionalidades novas.
- Não adicionar abas novas.
- Não trocar o conceito visual por layout clean/genérico.
- Não remover os corpos/estruturas corporais 3D quando fizerem parte da referência.
- Manter os botões e fluxos funcionando.

## Abas principais esperadas

- Início
- Treino
- Dieta
- Perfil

## Observação importante

A mudança não é apenas visual estática. Após aplicar o layout, auditar todos os botões, navegação, ações, telas, modais e fluxos para garantir que continuam funcionando.
