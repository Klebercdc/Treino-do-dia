# CLAUDE CODE — KRONIA DESIGN REPLICATION

OBJETIVO:

Replicar fielmente o estilo visual da referência enviada para dentro do aplicativo KRONIA real.

REGRAS:

- Não criar novo app.
- Não criar telas extras.
- Não criar novas funcionalidades.
- Não alterar arquitetura sem necessidade.
- Não substituir por design minimalista genérico.
- Replicar o estilo visual do ZIP.
- Adaptar o visual somente para telas existentes.
- Preservar fluxos reais.

OBRIGATÓRIO:

Após implementar:

1. Auditar navegação.
2. Auditar botões.
3. Auditar rotas.
4. Auditar estados vazios.
5. Auditar modais.
6. Auditar responsividade.
7. Auditar telas quebradas.
8. Garantir que layout novo não destruiu funcionalidades.

Critério final:

"Layout novo funcionando" > "Layout bonito parado"
